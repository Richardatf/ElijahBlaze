# ElijahBlaze.com

Public development hub for **Beneath Eden**, the Elijah Blaze Unity game, the Blaze Codex, and Elijah Blaze Studios.

## Canonical Avatar Asset

Place the supplied guardian image at:

```text
assets/elijah-blaze-avatar.png
```

The website and `AGENTS.md` both use that path as the canonical Elijah Blaze project avatar.

## Netlify

- Build command: leave blank
- Publish directory: `.`
- Production branch: `main`

Every push to `main` can then redeploy ElijahBlaze.com automatically.

## Codex

Codex should read `AGENTS.md` before modifying this repository. It contains the project premise, avatar definition, website standards, Unity rules, and the Havari Test.
